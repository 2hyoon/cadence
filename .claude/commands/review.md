Review the changes in this project.

First, read the following documents:
- `/CLAUDE.md`
- `/docs/ARCHITECTURE.md`
- `/docs/ADR.md`

Then check the changed files and verify them against the checklist below:

## Checklist

1. **Architecture compliance**: Does it follow the directory structure defined in ARCHITECTURE.md?
2. **Tech stack compliance**: Did it stay within the technical choices defined in the ADR?
3. **Tests present**: Are tests written for new features?
4. **CRITICAL rules**: Did it avoid violating the CRITICAL rules in CLAUDE.md?
5. **Buildable**: Does the build command pass without errors?

## Output Format

| Item | Result | Notes |
|------|------|------|
| Architecture compliance | ✅/❌ | {details} |
| Tech stack compliance | ✅/❌ | {details} |
| Tests present | ✅/❌ | {details} |
